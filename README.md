# 🚀 FEFE LAB – Frontend Experiments Collection

This repository contains a collection of frontend mini-projects and experiments built using modern web technologies like **HTML, CSS, JavaScript, and Vite**.

Each folder represents an individual project focusing on specific UI concepts and functionality.

---

## 📁 Project Structure

```
FEFE/
│
├── conditional-app/
├── dynamic-list-using-map/
├── image-slide/
├── jsx-components/
├── tracker/
├── user-form-input-submission/
```

---

## 📌 Projects Overview

### 1️⃣ Conditional App

* Demonstrates conditional rendering in JavaScript/React
* Shows how UI changes based on logic or state
* Useful for understanding dynamic UI behavior

---

### 2️⃣ Dynamic List Using Map

* Uses `.map()` to render lists dynamically
* Helps understand iteration in UI frameworks
* Common concept used in React applications

---

### 3️⃣ Image Slider

* Interactive image slideshow component
* Features:

  * Next / Previous navigation
  * Smooth transitions
* Useful for UI/UX design practice

---

### 4️⃣ JSX Components

* Basic reusable component structure
* Demonstrates:

  * Component-based architecture
  * Clean UI separation
* Foundation for React development

---

### 5️⃣ Tracker App

* Simple tracking system (could be expense/task tracker)
* Demonstrates:

  * State handling
  * User interaction
* Good practice for real-world app logic

---

### 6️⃣ User Form Input Submission

* Form handling and input validation
* Captures and processes user data
* Demonstrates:

  * Event handling
  * Form submission logic

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Vite (for fast development)
* Node.js (environment)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/R-GOPIRAM/FEFE_LAB.git
```

2. Navigate into a project folder:

```bash
cd conditional-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```
